Game of Life



If a cell is alive and has 2 or 3 neighbors, in the next generation it stays alive If a cell is alive and has 0 or 1 neighbors, it dies of loneliness If a cell is alive and has more than 3 neighbors, it dies of overpopulation If a cell is dead and has exactly 3 neighbors, it becomes alive in the next generation If a cell is dead and doesn't have exactly 3 neighbors, it stays dead


Press 'r' to randomly re-initialize the grid Press 'a' to see what happens if you make alive every 2nd cell in a row Press 'b' to see what happens if you make alive every 10th cell in a column Press 'c' to see what happens if you make alive every 50th cell in a row Press 'd' to make alive every 55th cell in a column and every 10th cell in a row Press 'e' to see what happens if you make alive every 35th cell in a row and column Press 'f' to see what happens if you make alive every cell in the grid Press 'q' to stop the game
